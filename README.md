# Sovereign ESG Scores and Bond Spreads

## Research Question
Do sovereign ESG scores predict government bond spreads?

## Methodology
**Language:** Python  
**Methods:** Panel fixed effects, World Bank WGI

## Data
World Bank WGI governance indicators (30 countries, 2005–2022)

## Key Findings
Within-country governance improvements significantly associated with lower sovereign bond spreads.

## How to Run
```bash
pip install -r requirements.txt
python code/project13_*.py
```
